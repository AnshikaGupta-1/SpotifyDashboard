# 🎵 Spotify Music Analysis Dashboard

An interactive Power BI dashboard that analyzes Spotify streaming performance across tracks, artists, release years, listening patterns, and audio characteristics.

The dashboard provides insights into streaming trends, artist performance, and song attributes using an intuitive and visually engaging interface inspired by Spotify's branding.

# Project Overview

This project explores Spotify streaming data to uncover patterns in music popularity and listener engagement. Using Power BI, the dashboard transforms raw streaming data into meaningful visualizations that allow users to analyze track performance, compare artists, explore release trends, and evaluate audio features.

The dashboard is fully interactive through slicers and cross-filtering, allowing users to dynamically explore the dataset.

---

# Business Objective

The objective of this dashboard is to help answer questions such as:

- Which songs generated the highest number of streams?
- Which artists consistently perform well?
- How have streaming trends changed over time?
- Which release periods produced the most streamed tracks?
- How do audio characteristics vary across popular songs?
- What listening patterns emerge throughout the week?

Such insights can assist music platforms, record labels, artists, and analysts in understanding audience preferences and content performance.

---

# Key Performance Indicators (KPIs)

The dashboard summarizes overall platform performance through KPI cards including:

- **Total Streams**
- **Total Tracks**
- **Average Streams per Track**

These metrics provide an instant overview before exploring detailed visualizations.

---

# Dashboard Features

## Release Trend Analysis

Visualizes streaming activity across release years, helping identify periods with increased music popularity and streaming growth.

---

## Monthly Performance

Displays:

- Number of tracks released each month
- Average streams generated per month

This helps identify seasonal publishing and streaming patterns.

---

## Daily Streaming Analysis

Shows cumulative streams distributed across the days of the week, highlighting listener behavior over time.

---

## Artist Performance

Selecting an artist dynamically updates:

- Most streamed track
- Album artwork
- Stream count
- Release date
- Musical key
- Mode
- Audio characteristics

---

## Top Performing Tracks

Displays the six highest streamed songs with album artwork, making it easier to compare the platform's biggest hits.

---

## Audio Feature Analysis

For the selected track, the dashboard displays important Spotify audio metrics including:

- Danceability
- Energy
- Speechiness
- Instrumentalness
- Liveness

These features help explain why certain songs may resonate more strongly with listeners.

---

# Interactive Filters

Users can filter the dashboard by:

- Artist
- Track
- Release Date

All visuals respond dynamically to filter selections through Power BI cross-filtering.
